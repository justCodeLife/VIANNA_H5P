# VIANNA_H5P
H5P implementation in C# and React !!!


# Features

Dark mode  
Translated all libraries to Persian  
Optimized assets loading for both editor and view  



# Usage

Back :

1) cd server
2) rename .env.example to .env & set environment variables  
3) run "dotnet watch run"  

Front :

1) cd client  
2) rename .env.example to .env & set environment variables  
3) run "npm start"
